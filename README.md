## 📖 Description

**Movie Ticket Booking System** is a beginner-to-intermediate level project that demonstrates core booking logic using **Java OOP** principles combined with a **front-end web interface** built using HTML and CSS. Users can browse currently showing movies, select a movie, choose the number of seats, and receive an instant total price calculation — all through a clean, dark-themed UI.

This project showcases the integration of Java backend logic (Object-Oriented Programming) with a simple web front-end, making it an ideal portfolio piece for demonstrating full-stack awareness and foundational programming skills.

---

## ✨ Features

- 🎥 **Movie Listings Page** — Browse currently showing movies with titles and ticket prices
- 🪑 **Seat Selection** — Choose the number of seats for your preferred movie
- 💰 **Automatic Price Calculation** — Total price is calculated instantly based on selected seats and movie price
- ✅ **Booking Confirmation** — Receive a clear booking confirmation message
- 🖤 **Dark Themed UI** — Sleek black-and-gold design for an immersive cinema feel
- ♨️ **Java OOP Backend** — Built using clean Object-Oriented Programming principles with `Movie`, `Booking`, and `Main` classes

## 🚀 Getting Started

### Prerequisites

- [Java JDK 8+](https://www.oracle.com/java/technologies/downloads/) installed
- Any modern web browser (Chrome, Firefox, Edge)
- [VS Code](https://code.visualstudio.com/) or any code editor (optional)

### Running the Web Interface

1. Clone the repository:
   ```bash
   git clone https://github.com/Gouri1897/movie-ticket-booking-system.git
   ```
2. Navigate to the project folder:
   ```bash
   cd movie-ticket-booking-system
   ```
3. Open `index.html` in your browser:
   - Double-click `index.html`, or
   - Use VS Code Live Server extension

### Running the Java Backend

1. Open a terminal in the project directory.
2. Compile the Java files:
   ```bash
   javac Main.java Movie.java Booking.java
   ```
3. Run the application:
   ```bash
   java Main
   ```
4. Follow the prompts to enter the number of seats and view the total price.
